# 即梦AI图片工作流自动化

## 完整流程

```
1. 生成脚本（Excel）
   ↓
2. 我自动获取即梦AI的图片URL
   ↓
3. 在网页显示让你点击选择
   ↓
4. 自动下载并插入Excel
```

---

## 当前可用方案

### 方案A：网页选择（推荐）

1. 你在即梦AI生成图片
2. 我获取所有图片URL
3. 在网页显示（每行4张缩略图）
4. 你点击选择
5. 自动插入Excel

### 方案B：批量下载后处理

1. 你在即梦AI点击"资产"
2. 批量下载所有图片到一个文件夹
3. 我扫描文件夹，按顺序匹配镜头
4. 自动插入Excel

---

## 自动化程度对比

| 步骤 | 方案A | 方案B |
|:---|:---:|:---:|
| 生图 | 手动 | 手动 |
| 获取URL | ✅ 自动 | ❌ |
| 选择图片 | ✅ 点击 | ❌ |
| 下载 | ✅ 自动 | ✅ 自动 |
| 插入Excel | ✅ 自动 | ✅ 自动 |

## 使用示例

用户：帮我生成Open宠物饮水机的视频脚本图片

我：
1. 生成28个镜头的脚本Excel
2. 指导你在即梦AI生成图片
3. 自动获取图片并显示在网页
4. 你点击选择
5. 自动插入Excel

---

## 技术限制

即梦AI当前限制：
- 无公开API
- 图片URL有过期时间
- 需要保持浏览器登录状态

如需完全自动化，建议：
1. 本地部署 Stable Diffusion / ComfyUI
2. 使用有API的Leonardo.ai
